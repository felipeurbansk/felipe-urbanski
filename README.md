## Setup

- Rename the .env-example file to .env and change the settings if necessary

## Quick Start

First run this command in the project root folder to raise the mongodb container

```sh
docker-compose up -d
```

Install all project dependencies

```sh
npm install

or

yarn
```

Run the project

```sh
npm run dev

or

yarn dev
```
